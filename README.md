to use it quickly:
Save your file: Copy the code (index.html)  into Notepad and save it on your computer as index.html.

Go to tiiny.host.

Drag and drop your index.html file into the box on their homepage.

Enter a name for your site (e.g., hebrew-braille-app).

Click "Launch": They will give you a link immediately (like hebrew-braille-app.tiiny.site).
